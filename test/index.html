Rules of game

- There are 3 level : Easy, Medium, Hard.

   + Easy: There are 12 cards. Each false guess, You are subtracted 1 point. Win level, You receive 10 points


   + Medium: There are 18 cards. Each false guess, You are subtracted 2 point. Win level, You receive 20 points


   + Hard: There are 18 cards. Each false guess, You are subtracted 5 point. Win level, You receive 50 points


- Click save button to sign up and save point and click continue button to login.


- Click rank button to see rank.


- To have avatar, you must copy avatar to folder "avatar" in project and image must be jpg format. ( name of avartar must be the same as username )
